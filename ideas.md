# X News AI Dashboard - 디자인 브레인스토밍

## 프로젝트 개요
X(트위터) 뉴스를 자동 수집하고 Gemini AI로 분석하여 실시간으로 보여주는 웹 대시보드.
지정학, 경제, 트럼프, 암호화폐 4개 카테고리로 분류된 뉴스를 시각화하고, AI 분석 결과와 시장 영향도를 한눈에 확인.

## Idea 1: "Command Center" — 밀리터리 인텔리전스 스타일

<text>
### Design Movement
밀리터리/인텔리전스 커맨드 센터에서 영감을 받은 다크 테마 대시보드. 실시간 정보 모니터링 시스템의 긴장감과 전문성을 웹에 구현.

### Core Principles
1. **정보 밀도 극대화:** 한 화면에 최대한 많은 정보를 효율적으로 배치
2. **계층적 긴급성:** 뉴스의 중요도와 긴급성에 따라 시각적 강도를 차등 적용
3. **실시간 감시 분위기:** 데이터가 살아 움직이는 느낌의 인터페이스
4. **기능 우선 미학:** 장식보다 정보 전달 효율에 집중

### Color Philosophy
- **배경:** 깊은 네이비/차콜 (#0a0f1a ~ #111827) — 장시간 모니터링에 적합한 눈의 피로 최소화
- **주요 액센트:** 사이버 그린 (#00ff88) — 활성 데이터, 실시간 업데이트 표시
- **경고:** 앰버 (#fbbf24) — 주의 필요 뉴스
- **위험:** 레드 (#ef4444) — 긴급/고영향 뉴스
- **보조:** 쿨 블루 (#3b82f6) — 일반 정보, 링크

### Layout Paradigm
- 3단 비대칭 그리드: 좌측 좁은 카테고리 네비게이션 + 중앙 넓은 뉴스 피드 + 우측 AI 분석 패널
- 상단 고정 바에 실시간 시스템 상태 표시 (마지막 업데이트, 수집된 트윗 수, AI 분석 상태)
- 하단에 뉴스 티커 스트립

### Signature Elements
1. **글로우 보더:** 카드 테두리에 카테고리별 색상의 은은한 글로우 효과
2. **스캔라인 오버레이:** 배경에 미세한 스캔라인 텍스처로 모니터 느낌
3. **펄스 인디케이터:** 실시간 데이터 수신 시 맥박처럼 뛰는 도트 애니메이션

### Interaction Philosophy
- 호버 시 카드가 살짝 밝아지며 추가 정보 레이어 노출
- 카테고리 전환 시 데이터가 터미널 타이핑처럼 순차 로딩
- 스크롤 시 뉴스 카드가 레이더 스캔처럼 순차적으로 나타남

### Animation
- 뉴스 카드 진입: 좌측에서 슬라이드인 + 페이드인 (stagger 50ms)
- 실시간 업데이트: 새 뉴스 도착 시 상단에서 드롭다운 + 글로우 펄스
- 시스템 상태 바: 숫자 카운터 롤링 애니메이션
- 배경: 미세한 그리드 패턴이 천천히 이동하는 패럴랙스

### Typography System
- **헤드라인:** Space Grotesk Bold — 기술적이면서 가독성 높은 산세리프
- **본문:** IBM Plex Sans Regular — 데이터 친화적, 숫자 가독성 우수
- **모노스페이스 데이터:** JetBrains Mono — 시간, 수치 등 고정폭 데이터
</text>

## Idea 2: "Bloomberg Terminal Meets Editorial" — 금융 뉴스룸 스타일

<text>
### Design Movement
Bloomberg Terminal의 데이터 밀도와 Financial Times의 에디토리얼 품격을 결합한 하이브리드 디자인. 전문 투자자/트레이더가 신뢰할 수 있는 권위 있는 뉴스 인터페이스.

### Core Principles
1. **에디토리얼 권위:** 신문 편집실의 전통적 위계와 현대 데이터 시각화의 결합
2. **스캔 가능성:** 빠른 눈 훑기로 핵심 정보를 즉시 파악 가능한 구조
3. **데이터와 내러티브의 공존:** 숫자와 이야기가 자연스럽게 어우러지는 레이아웃
4. **절제된 우아함:** 과도한 장식 없이 타이포그래피와 여백으로 품격 표현

### Color Philosophy
- **배경:** 오프화이트 (#fafaf9) — 신문지의 따뜻한 질감 연상
- **텍스트:** 딥 차콜 (#1c1917) — 강한 대비로 가독성 극대화
- **주요 액센트:** FT 핑크/살몬 (#f5c6aa) — Financial Times 오마주, 프리미엄 느낌
- **카테고리 색상:** 지정학(딥레드 #991b1b), 경제(네이비 #1e3a5f), 트럼프(골드 #b45309), 암호화폐(에메랄드 #047857)
- **구분선:** 연한 그레이 (#e7e5e4) — 섹션 구분

### Layout Paradigm
- 신문 스타일 멀티 컬럼 레이아웃: 메인 헤드라인 영역(전체 폭) + 2~3단 뉴스 그리드
- 좌측 사이드바에 카테고리 필터 + AI 분석 요약 위젯
- 상단에 "오늘의 브리핑" 히어로 섹션 — 가장 중요한 뉴스 1개를 대형으로 배치
- 우측에 "시장 영향도" 미니 차트 위젯

### Signature Elements
1. **세리프 + 산세리프 타이포 믹스:** 헤드라인은 세리프로 권위감, 본문은 산세리프로 현대감
2. **카테고리 컬러 바:** 각 뉴스 카드 좌측에 카테고리 색상의 얇은 세로 바
3. **"BREAKING" 배지:** 긴급 뉴스에 빨간 배지 + 미세한 펄스 애니메이션

### Interaction Philosophy
- 호버 시 카드 배경이 미세하게 어두워지며 그림자 깊어짐
- 카테고리 필터 클릭 시 뉴스가 매거진 페이지 넘기듯 전환
- AI 분석 섹션 확장 시 아코디언 방식으로 부드럽게 열림

### Animation
- 페이지 로드: 신문 인쇄처럼 위에서 아래로 순차적 페이드인
- 뉴스 카드: 호버 시 0.2s ease-out으로 그림자 + 스케일(1.01) 전환
- 카테고리 전환: 컨텐츠 영역만 크로스페이드 (0.3s)
- 숫자 데이터: 카운트업 애니메이션으로 동적 느낌

### Typography System
- **메인 헤드라인:** Playfair Display Bold — 클래식한 세리프, 뉴스 권위감
- **서브 헤드라인:** Noto Sans KR Medium — 한국어 최적화 산세리프
- **본문:** Noto Sans KR Regular — 한국어 가독성 최우선
- **데이터/수치:** Tabular Lining의 DM Sans — 숫자 정렬 최적화
</text>

## Idea 3: "Neon Pulse" — 사이버펑크 데이터 스트림 스타일

<text>
### Design Movement
사이버펑크 + 글리치 아트에서 영감을 받은 초현대적 대시보드. 데이터가 네온 빛 속에서 흐르는 듯한 미래지향적 인터페이스. Blade Runner와 Tron의 시각 언어를 웹에 구현.

### Core Principles
1. **데이터 스트림 시각화:** 뉴스를 정적 카드가 아닌 흐르는 데이터 스트림으로 표현
2. **네온 계층 구조:** 밝기와 색상의 네온 강도로 정보 위계 표현
3. **글리치 미학:** 의도적 글리치 효과로 실시간 데이터의 역동성 표현
4. **몰입형 경험:** 사용자가 데이터 세계 속에 들어와 있는 듯한 공간감

### Color Philosophy
- **배경:** 순수 블랙에 가까운 다크 (#050510) — 네온 색상의 극대화된 대비
- **주요 네온:** 일렉트릭 시안 (#00f0ff) — 주요 데이터, 활성 요소
- **보조 네온:** 핫 마젠타 (#ff00aa) — 경고, 하이라이트
- **삼차 네온:** 라임 그린 (#aaff00) — 긍정적 지표, 성공 상태
- **텍스트:** 쿨 화이트 (#e0e8ff) — 네온 배경에서의 가독성

### Layout Paradigm
- 풀스크린 몰입형: 네비게이션 최소화, 컨텐츠 영역 극대화
- 중앙에 대형 "메인 피드" — 뉴스가 위에서 아래로 스트리밍
- 좌우 사이드에 부유하는 위젯들 (AI 분석, 카테고리 필터, 시스템 상태)
- 배경에 움직이는 그리드/파티클 효과

### Signature Elements
1. **네온 글로우 텍스트:** 주요 헤드라인에 CSS text-shadow로 네온 사인 효과
2. **글리치 트랜지션:** 페이지 전환 시 0.1초간 RGB 분리 글리치 효과
3. **데이터 레인:** 배경에 Matrix 스타일의 떨어지는 데이터 문자열

### Interaction Philosophy
- 호버 시 요소가 네온 빛으로 "활성화"되는 느낌
- 클릭 시 짧은 글리치 + 확장 애니메이션
- 스크롤 시 배경 파티클이 반응하여 움직임

### Animation
- 뉴스 카드 진입: 글리치 플리커 + 페이드인 (0.4s)
- 네온 글로우: 지속적인 미세 펄스 (2s infinite)
- 배경 그리드: 느린 회전 + 원근 이동 (60s infinite)
- 데이터 업데이트: 숫자가 슬롯머신처럼 롤링

### Typography System
- **헤드라인:** Orbitron Bold — 미래지향적 기하학 폰트
- **본문:** Rajdhani Medium — 테크니컬하면서 가독성 있는 산세리프
- **데이터:** Share Tech Mono — 터미널/코딩 느낌의 모노스페이스
</text>
